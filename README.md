## Справочник сотрудников (Python + SQLite)

Консольное приложение с 6 режимами работы: создание таблицы, добавление записи, вывод уникальных строк, массовая генерация и вставка, замер времени запроса, оптимизация индексацией.

### Требования
- Python 3.10+
- Зависимости: стандартная библиотека (`sqlite3`), внешние пакеты не нужны

### Установка
Windows PowerShell:
```
cd "C:\Users\amiri\PycharmProjects\Task_2"
python --version
# при необходимости используйте: py --version
```

### Запуск (режимы)
- 1) Создать таблицу:
```
python main.py 1
```

- 2) Добавить одну запись:
```
python main.py 2 "Ivanov Petr Sergeevich" 2009-07-12 Male
```

- 3) Вывести уникальные строки (ФИО+дата), отсортированные по ФИО, с возрастом:
```
python main.py 3
```

- 4) Сгенерировать 1 000 000 записей (+100 Male с фамилией на 'F') и вставить пакетно:
```
python main.py 4
```

- 5) Выполнить запрос (Male, фамилия начинается с 'F') и замерить время:
```
python main.py 5
```

- 6) Создать индексы для оптимизации и повторно замерить время:
```
python main.py 6
```

### Структура
```
Task_2/
  main.py
  src/
    __init__.py
    db.py
    models.py
    utils.py
  data/
    employees.sqlite  (создаётся автоматически)
```

### Примечания
- БД: `data\employees.sqlite`
- Пол: `Male` или `Female`
- Формат даты: `YYYY-MM-DD`
- Для «чистого» старта удалите файл БД и выполните режим 1
